== Description ==
> This plugin requires the <a href="https://www.closemarketing.es/likes/genesis/">Genesis Framework</a>. <strong>Don't use Genesis Framework? <a href="https://www.closemarketing.es/likes/genesis/">Buy the framework</a></strong> and start using this revolutionary framework!

This plugin give you a bunch of widgets that allows you to draw content using Genesis and SiteOrigin Page Builder.

Widgets included in this plugin:

-   Social Icons.
-   Contact information.
-   Child menu for pages. Shows child pages in sidebar in active page.
-   Button for CTA with Genesis css classes.
-   WooCommerce Category Image in a genesis grid.
-   Buttons for iOS apps and Android Apps.
-   Youtube and third party embed video.

For WooCommerce:
-   Category image.
-   Featured products.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your
WordPress installation and then activate the Plugin from Plugins page.
